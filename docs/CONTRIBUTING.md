# How to Contribute

To contribute to ConfigFinder, please check our [complete contribution guide](https://github.com/EPIHACKBENIN/ConfigFinder/blob/main/CONTRIBUTING.md).

## Quick Links

- [Project Repository](https://github.com/EPIHACKBENIN/ConfigFinder)
- [Open Issues](https://github.com/EPIHACKBENIN/ConfigFinder/issues)
- [Pull Requests](https://github.com/EPIHACKBENIN/ConfigFinder/pulls)

## Quick Start

1. Fork the repository
2. Create a branch for your changes
3. Make your changes
4. Test locally with `mkdocs serve`
5. Submit a Pull Request

Need help? Contact us at epihack.benin@gmail.com
